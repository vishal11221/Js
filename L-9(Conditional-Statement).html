In javascript three type of Conditional Statement.
  1. if:-In if block statement run when if block condition statisfied.
  2. if else ,if block statement run if your condition true otherwise your else block statement run.
  3. if elseif, if block statement run if your condition true, then it goes to other if block,check that block if that block is not statisfied then it goes to other if block if it is also not true,then it goes to else block.(it is use for check multiple condition).
