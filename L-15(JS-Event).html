Event handling the action that we perform on web-pages.
  1. Mouse-Event:-The Event occur from mouse press that are called Mouse-Event like:- click,doubleclick, rightclick,mousedown,mouseover
    1. Click
    2. Right-Click
    3. Double-Click
    4. mouseover
    5. mouseout
    6. mouseup
    7. mousedown
  2. Keyboard-Event:- The Event occur from keyboard press that are called Keyboard-Event like:-keypress,keyup and so on.
    1. keypress
    2. keyup
  3. Window Event/Document Event:-The Event occur when play with window like resize,load,unload and so on.
    1. load
    2. unload
    3. resize
    4. scroll
  4. Form Event:-The Event occur when changes in forms input area like:-onblur,onfocus,onsubmit and so on.
    1. blur
    2. focus
    3. submit
    4. reset
