  2. do while-loop:-it is a type of loop it execute first statement then check the condition,if condition not statisfied it not execute our statement.
  3. for Loop:-It is also a type of loop that is totaly different by there syntax,in one line we intilize the value,apply the condition and increment the value.
  4. for in Loop:-It is used to print the Object value.
