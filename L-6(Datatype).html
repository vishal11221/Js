1. Primitive Datatypes:-
    1. String:-it define with single,double and backticks.it store character values
    2. Number:-it define the numeric value as well as float value also.
    3. Bollean:- it define only true and false value.
    4. Null:-it store empty value.
    5. Undefined:-When we declare variable but not store any value in it.
  2. Non-Primitive Datatypes:-
    1. Array:-it store multiple type of data in single variable but in Array format,We see the Array value with the help of foreach loop
    2. Object:-it store multiple type of data in single variable but in object form,We see the Object value with the help of forin loop
    3. Function:-it is set of block of code that is used when we call it.
  3. We also find the datatype of variables with "type of" operator.
    as:- typeOf 10,it return Number
  4. Why javascript also called dynamic language ?
    JavaScript is dynamic typed language because in js no need to declare the datatype of variables.
  5. Is javascript is statically typed language or dynamic typed language ?
    Js is dynamically typed language but Typescript is statically type language.
    In dynamically typed language all the type checks are performed in a runtime,only when your program is executing,so this means you can just assign anything you want to the variable and it will work.
    If we take a look at Typescript,it is a statically typed language,So all checks will be performed during compile.
  6. JavaScript is Cross-platform or not ?
    Js is Cross-platform,we run js to any platform like,MAC,Windows,linux and so on.
