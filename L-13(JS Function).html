Function is a block of code that is used when we call it.

Function remove the reptation from your code.

    Type of Function:-
  1. Function with Parameter:-Function with parameter that means,we passed the variable at the time of function declaration and passed that parameter value as an argument at the time of function calling.
  2. Default Parameter Value:-Es6 allows function parameter to have default values.We set default value at the time of parameter pass through function.We also change the default value to assign default value in argument.
  3. Rest Parameter:- The rest parameter allows a function to treat an indefinite number of argument as an array.
  4. Function with return value:- function that has return keyword to return some value.
